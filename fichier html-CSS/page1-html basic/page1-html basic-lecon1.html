Mini Leçon : Les Bases de l'HTML

1. Structure de Base d'une Page HTML

Voici la structure de base d'une page HTML :

<!DOCTYPE html>
<html>
  <head>
    <title>Ma première page</title>
  </head>
  <body>
    <h1>Bienvenue sur ma page</h1>
    <p>Ceci est un paragraphe.</p>
  </body>
</html>

Chaque document commence par <!DOCTYPE html>, suivi des balises <html>, <head>, et <body>.

2. Les Balises Essentielles

Voici quelques balises essentielles :

  • Titres : <h1> à <h6> (du plus grand au plus petit) :
    • <h1>Titre principal</h1>
<h2>Sous-titre</h2>
  • Paragraphes : <p> :
    • <p>Ceci est un paragraphe.</p>
  • Liens : <a> :
    • <a href="https://example.com">Clique ici</a>
  • Images : <img> :
    • <img src="image.jpg" alt="Description de l'image">

3. Les Attributs

Les balises peuvent avoir des attributs pour ajouter des informations supplémentaires :

  • href : Utilisé pour les liens (<a>).
  • src : Spécifie la source d'une image (<img>).
  • alt : Ajoute une description alternative pour les images.
  • style : Permet d'ajouter du style directement dans une balise.

Exemple d'attributs dans une balise :

<a href="https://example.com" target="_blank">Visitez un site</a>

4. Hiérarchie et Organisation

Structurez votre page avec une hiérarchie claire :

<h1>Page principale</h1>
<h2>Section 1</h2>
<p>Texte de la première section.</p>
<h2>Section 2</h2>
<p>Texte de la deuxième section.</p>
