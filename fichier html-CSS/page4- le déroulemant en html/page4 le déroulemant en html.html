Petite leçon : Les éléments déroulants en HTML

1) Le menu déroulant (select)

C'est un menu où l'utilisateur choisit une option.

<select>
  <option>Option 1</option>
  <option>Option 2</option>
  <option>Option 3</option>
</select>

2) Le bandeau déroulant (marquee)

Texte qui défile automatiquement.

<marquee>Texte qui défile</marquee>

3) Le bloc déroulant (scroll)

Un bloc qui affiche une barre de défilement quand il y a trop de contenu.

<div style="height:100px; overflow:auto;">
  Long contenu...
</div>

4) La zone de texte déroulante (textarea)

Une zone d'écriture qui scrolle si le texte est trop long.

<textarea rows="4">Texte...</textarea>

5) Le bloc qui se déplie (details + summary)

Un élément qui s'ouvre quand on clique dessus.

<details>
  <summary>Cliquer pour ouvrir</summary>
  Contenu caché.
</details>

Résumé

  • <select> : menu déroulant
  • <marquee> : texte qui défile
  • <div overflow> : bloc scrollable
  • <textarea> : zone scrollable
  • <details> : bloc qui se déplie
