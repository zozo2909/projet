Leçon 6 : Les tableaux

Les tableaux permettent d’organiser les informations en lignes et colonnes.

Exemple de tableau

<table border="1">
<tr>
<th>Nom</th>
<th>Âge</th>
</tr>
<tr>
<td>Lucas</td>
<td>18</td>
</tr>
<tr>
<td>Emma</td>
<td>17</td>
</tr>
</table>

Explications :

  • <table> : crée le tableau
  • <tr> : ligne
  • <th> : titre de colonne
  • <td> : cellule normale
