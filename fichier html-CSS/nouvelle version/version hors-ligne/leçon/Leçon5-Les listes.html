Les listes

HTML propose 4 types de listes pour organiser des éléments de manière claire.

Liste à puces — <ul>

Utilisée quand l'ordre n'a pas d'importance.

<ul>
  <li>Pomme</li>
  <li>Banane</li>
  <li>Orange</li>
</ul>

Liste numérotée — <ol>

Utilisée pour des étapes, un classement, une procédure.

<ol>
  <li>Préparer les ingrédients</li>
  <li>Mélanger</li>
  <li>Cuire 20 min</li>
</ol>

<!-- Options de numérotation -->
<ol start="5">...</ol>           <!-- commence à 5 -->
<ol reversed>...</ol>         <!-- numérotation inversée -->
<ol type="A">...</ol>          <!-- A, B, C... -->

Listes imbriquées

<ul>
  <li>Fruits
    <ul>
      <li>Pomme</li>
      <li>Banane</li>
    </ul>
  </li>
  <li>Légumes</li>
</ul>

Liste de définitions — <dl>

Paires terme/définition. Utile pour des glossaires.

<dl>
  <dt>HTML</dt>
  <dd>Langage de structuration du web.</dd>
  <dt>CSS</dt>
  <dd>Langage de mise en forme.</dd>
</dl>

Bonnes pratiques

  • Choisir le bon type : <ul> si l'ordre n'importe pas, <ol> pour des étapes.
  • Ne mettre que des <li> comme enfants directs de <ul> et <ol>.
  • Les menus de navigation sont des <ul> contenant des liens.
