Leçon 5 : Les listes

Les listes permettent de présenter plusieurs éléments de manière claire.

Liste à puces (non ordonnée)

<ul>
<li>Pomme</li>
<li>Banane</li>
<li>Orange</li>
</ul>

Liste numérotée (ordonnée)

<ol>
<li>Étape 1 : Préparer</li>
<li>Étape 2 : Mélanger</li>
<li>Étape 3 : Cuire</li>
</ol>

Listes imbriquées

<ul>
<li>Fruits
  <ul>
    <li>Pomme</li>
    <li>Banane</li>
  </ul>
</li>
</ul>
