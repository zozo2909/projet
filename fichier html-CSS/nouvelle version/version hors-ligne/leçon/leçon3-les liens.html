Les liens hypertexte

Un lien est créé avec la balise <a> (anchor). L'attribut href contient la destination.

<a href="destination">Texte cliquable</a>

Les 4 types de liens

<!-- Lien interne (autre page du site) -->
<a href="contact.html">Page Contact</a>

<!-- Lien externe (autre site) -->
<a href="https://wikipedia.org">Wikipedia</a>

<!-- Lien ancre (section de la même page) -->
<a href="#section-contact">Aller au contact</a>

<!-- Lien email -->
<a href="mailto:[email protected]">Nous écrire</a>

<!-- Lien téléphone (mobile) -->
<a href="tel:+33612345678">06 12 34 56 78</a>

Les attributs de <a>

AttributValeursDescription
hrefURL, chemin, #id, mailto:Destination du lien
target_blank, _self_blank = nouvel onglet
relnoopener, noreferrerSécurité avec target="_blank"
titleTexteInfo-bulle au survol
downloadnom-fichierForce le téléchargement
<!-- Ouvrir dans un nouvel onglet — TOUJOURS ajouter rel -->
<a href="https://exemple.com"
   target="_blank"
   rel="noopener noreferrer">Visiter</a>

<!-- Téléchargement -->
<a href="guide.pdf" download="guide-html.pdf">Télécharger</a>
Avec target="_blank", toujours ajouter rel="noopener noreferrer". Sans ça, la page ouverte peut accéder à votre page via JavaScript.

Chemins relatifs vs absolus

<!-- Même dossier -->
<a href="contact.html">Contact</a>

<!-- Sous-dossier -->
<a href="pages/contact.html">Contact</a>

Bonnes pratiques

  • Toujours ajouter rel="noopener noreferrer" avec target="_blank".
  • Le texte du lien doit être descriptif — éviter "cliquez ici".
  • Utiliser des chemins relatifs pour les liens internes.
