Leçon 10 : Balises sémantiques

Les balises sémantiques donnent du sens au contenu pour le navigateur et les moteurs de recherche.

Exemple complet

<header>
<h1>Mon site HTML</h1>
</header>

<nav>
<a href="index.html">Accueil</a>
<a href="contact.html">Contact</a>
</nav>

<main>
<p>Contenu principal de la page.</p>
</main>

<footer>
<p>Copyright 2026</p>
</footer>

Explications :

  • header : en-tête de la page
  • nav : menu de navigation
  • main : contenu principal
  • footer : pied de page
