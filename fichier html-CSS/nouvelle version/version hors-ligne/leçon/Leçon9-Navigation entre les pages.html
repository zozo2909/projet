Navigation entre les pages

Un site web est un ensemble de fichiers HTML reliés entre eux. La page d'accueil s'appelle toujours index.html.

Structure d'un site multi-pages

mon-site/
├── index.html       ← Page d'accueil
├── contact.html
├── apropos.html
├── images/
└── css/

La balise <nav>

La balise <nav> indique au navigateur et aux lecteurs d'écran que son contenu est un menu de navigation.

<nav>
  <ul>
    <li><a href="index.html">Accueil</a></li>
    <li><a href="apropos.html">À propos</a></li>
    <li><a href="contact.html">Contact</a></li>
  </ul>
</nav>
La structure recommandée est <nav> → <ul> → <li> → <a>. Un menu est sémantiquement une liste de liens.

Indiquer la page courante

<!-- Via une classe CSS -->
<a href="index.html" class="active">Accueil</a>

<!-- Via aria-current (accessibilité) -->
<a href="index.html" aria-current="page">Accueil</a>

Navigation intra-page (ancres)

<!-- Sommaire -->
<nav>
  <a href="#introduction">Introduction</a>
  <a href="#conclusion">Conclusion</a>
</nav>

<!-- Sections cibles -->
<section id="introduction">
  <h2>Introduction</h2>
</section>

Fil d'Ariane (breadcrumb)

<nav aria-label="Fil d'Ariane">
  <ol>
    <li><a href="/">Accueil</a></li>
    <li><a href="/blog">Blog</a></li>
    <li aria-current="page">Article du jour</li>
  </ol>
</nav>

Bonnes pratiques

  • Toujours nommer la page d'accueil index.html.
  • Utiliser des chemins relatifs pour les liens internes.
  • Indiquer la page courante avec aria-current="page".
  • La navigation doit être cohérente sur toutes les pages.
