Structure d'une page HTML

HTML (HyperText Markup Language) est le langage de base du web. Il structure le contenu : titres, paragraphes, images, liens... C'est le squelette sur lequel tout repose. CSS s'occupe du style, JavaScript des interactions.

Les deux parties d'une page HTML

  1. <head> — Zone invisible. Titre de l'onglet, encodage, liens CSS, métadonnées.
  2. <body> — Zone visible. Tout ce que l'utilisateur voit à l'écran.

Exemple complet

<!-- Déclaration obligatoire en première ligne -->
<!DOCTYPE html>
<html lang="fr">
  <head>
    <meta name="viewport" content="width=device-width">
    <title>Mon premier site</title>
  </head>
  <body>
    <h1>Bienvenue !</h1>
    <p>Ma première page HTML.</p>
  </body>
</html>

Explication ligne par ligne

  • <!DOCTYPE html> — Indique que la page utilise HTML5. Toujours en première ligne.
  • <html lang="fr"> — Balise racine. L'attribut lang précise la langue pour les lecteurs d'écran et Google.
  • <meta name="viewport"> — Rend la page adaptée aux mobiles.
  • <title> — Texte affiché dans l'onglet du navigateur et dans les résultats Google.

Comment fonctionne une balise ?

Une balise HTML s'écrit entre chevrons. La plupart vont par paires :

<balise> contenu </balise>

<!-- Certaines sont auto-fermantes -->
<img src="photo.jpg">
<br>   <!-- saut de ligne -->
<hr>   <!-- ligne horizontale -->
Les balises doivent être correctement imbriquées : si on ouvre <div> puis <p>, on ferme </p> avant </div>.

Bonnes pratiques

  • Toujours commencer par <!DOCTYPE html>.
  • Toujours ajouter lang="fr".
  • Indenter le code pour le rendre lisible.
  • Commenter les parties complexes avec <!-- commentaire -->.
