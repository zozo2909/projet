Leçon 2 : Les balises de texte

HTML utilise des balises pour organiser le texte et lui donner un sens. Chaque balise indique au navigateur ce que représente le texte.

Les titres

Les titres permettent de hiérarchiser les informations :

  • <h1> : titre principal (un seul par page)
  • <h2> : sous-titre
  • <h3> à <h6> : sous-sections

Exemple de titres et paragraphes

<h1>Mon site web</h1>
<h2>Présentation</h2>
<p>Voici un paragraphe d'explication.</p>
<h3>Détails</h3>
<p>Encore plus d'informations ici.</p>

Texte important ou emphase

Pour mettre en valeur du texte :

  • <strong>texte</strong> : texte important (gras)
  • <em>texte</em> : texte en emphase (italique)
