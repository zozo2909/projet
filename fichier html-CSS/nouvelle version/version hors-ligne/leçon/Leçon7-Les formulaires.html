Leçon 7 : Les formulaires

Les formulaires permettent aux utilisateurs de fournir des informations (nom, email, etc.).

Exemple de formulaire simple

<form>
<label>Nom :</label>
<input type="text" name="nom">
<label>Email :</label>
<input type="email" name="email">
<button type="submit">Envoyer</button>
</form>

Explications :

  • <input type="text"> : champ pour écrire du texte
  • <input type="email"> : champ pour écrire un email
  • <button type="submit"> : bouton pour envoyer les données
