Leçon 1 : Structure d’une page HTML

HTML (HyperText Markup Language) est le langage qui sert à structurer une page web. On peut le comparer à un squelette : il donne une forme au site, mais ne s’occupe pas encore du style.

Structure de base d’une page HTML

Une page HTML se divise toujours en deux parties principales :

  1. <head> : zone invisible contenant les informations techniques (titre, encodage des caractères, etc.)
  2. <body> : zone visible où se trouve tout le contenu affiché à l’écran.

Exemple d’une page HTML complète

<!DOCTYPE html>
<html lang="fr">
<head>
<meta charset="UTF-8">
<title>Mon premier site</title>
</head>
<body>
<h1>Bienvenue !</h1>
<p>>Ceci est ma première page HTML.</p>
</body>
</html>

Explications

  • <!DOCTYPE html> : indique au navigateur que c’est du HTML5.
  • <html lang="fr"> : indique que la langue est le français.
  • <head> : contient les informations techniques.
  • <body> : contient le contenu visible.

Bonnes pratiques

  • Commencer toujours par <!DOCTYPE html>.
  • Mettre la langue de la page avec lang="fr".
  • Séparer clairement <head> et <body>.
