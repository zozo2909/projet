Leçon 4 : Les images

Les images rendent un site plus attractif et permettent d’illustrer le contenu.

Image simple

<img src="chat.jpg" alt="Photo d'un chat">

Explications :

  • src : chemin vers le fichier image.
  • alt : texte alternatif si l’image ne s’affiche pas.

Redimensionner l’image
