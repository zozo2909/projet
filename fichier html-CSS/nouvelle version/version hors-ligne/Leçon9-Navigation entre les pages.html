Leçon 9 : Navigation entre les pages

Un site est souvent composé de plusieurs pages reliées entre elles par des liens.

Exemple de menu de navigation

<nav>
<a href="index.html">Accueil</a>
<a href="contact.html">Contact</a>
<a href="about.html">À propos</a>
</nav>

Explications :

  • index.html : page principale
  • Les liens permettent de naviguer facilement entre les pages.
