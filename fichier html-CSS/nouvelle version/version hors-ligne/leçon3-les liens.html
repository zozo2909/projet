Leçon 3 : Les liens

Les liens (<a>) permettent de naviguer entre les pages d’un site ou vers d’autres sites web.

Lien interne

Vers une autre page du site :

<a href="contact.html">Page Contact</a>

Lien externe

Vers un autre site :

<a href="https://wikipedia.org">Wikipedia</a>

Lien ouvrant dans un nouvel onglet

<a href="https://wikipedia.org" target="_blank">Wikipedia</a>

Explication :

  • href : indique l’adresse du lien.
  • target="_blank" : ouvre le lien dans un nouvel onglet.
