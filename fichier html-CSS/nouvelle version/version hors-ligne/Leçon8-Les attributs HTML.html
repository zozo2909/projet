Leçon 8 : Les attributs HTML

Les attributs donnent des informations supplémentaires aux balises.

Exemple

<p id="intro" class="important">Bonjour !</p>

Explications :

  • id : identifiant unique pour un élément
  • class : groupe d’éléments similaires
  • D’autres attributs : src, href, alt, name, type
