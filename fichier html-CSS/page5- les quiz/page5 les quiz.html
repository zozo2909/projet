Leçon : Créer un Quiz en HTML

1) Structure de base

Un quiz utilise un formulaire (<form>) et des questions avec des boutons radio (<input type="radio">).

<form id="quizForm">
  <div class="question">
    <p>1. Exemple de question ?</p>
    <label><input type="radio" name="q1" value="wrong"> Mauvaise réponse</label><br>
    <label><input type="radio" name="q1" value="correct"> Bonne réponse</label>
  </div>
  <button type="button" onclick="checkAnswers()">Voir mon score</button>
</form>

2) Ajouter plusieurs questions

Chaque question est dans un <div class="question"> et possède un nom unique (name="q1").

<div class="question">
  <p>2. Type de question ouverte ?</p>
  <label><input type="radio" name="q2" value="wrong"> QCM</label><br>
  <label><input type="radio" name="q2" value="correct"> Question ouverte</label>
</div>

3) Calculer le score

Le score se calcule en JavaScript en vérifiant quelles réponses sont correctes :

function checkAnswers() {
  const form = document.getElementById('quizForm');
  const result = document.getElementById('result');
  let score = 0;

  if (form.q1.value === "correct") score++;
  if (form.q2.value === "correct") score++;

  result.textContent = `Ton score est de ${score} sur 2 !`;
}

5) Résumé

  • Chaque question est dans un <div class="question">.
  • Chaque question a un nom unique (name="q1").
  • Marquer les réponses correctes avec value="correct".
  • Utiliser JavaScript pour calculer le score et l’afficher.
  • CSS permet de rendre le quiz joli et lisible.
