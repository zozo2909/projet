1. Les images

Pour afficher une image, on utilise la balise :

<img src="chemin d'accés pour l'image">

Explications :

  • src indique le chemin d'accés de l'image.
  • alt décrit l’image (utile si elle ne s’affiche pas).

3. Les boutons

Les boutons servent à faire une action sur la page.

<button>Clique ici</button>

Exemple avec une petite action :

<button onclick="alert('je suis un bouton')">bouton</button>

Différence entre un lien et un bouton :

  • Un lien sert à vous envoyer sur un autre site.
  • Un bouton sert à effectuer une action qui peut ouvrir un fichier, envoyer sur internet, contacter une personne.
