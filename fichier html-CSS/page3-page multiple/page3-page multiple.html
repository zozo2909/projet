Leçon HTML : les onglets et les redirections

1. Créer plusieurs pages

Chaque page de ton site est un fichier HTML :

login.html
home.html
page1.html

Pour naviguer entre les pages, on utilise le code :

<a href="(chemin d'accès du fichier)"title="Aller à l'accueil">

2. Transformer une image en bouton

Pour qu’une image devienne cliquable :

<a href="contact.html">
  <img src="mon-bouton.png" alt="Contact" width="150">
</a>

Exemple :

Tu peux ajouter un effet au survol :

img:hover {
  cursor: pointer;
  opacity: 0.8;
}
